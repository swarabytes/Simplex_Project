About OptiSolve

A smarter interface for solving linear optimization problems.

OptiSolve is a web-based mathematical optimization assistant designed to solve Linear Programming Problems (LPP) using the Simplex Method and Big-M Method.

It helps users define objective functions, enter constraints, solve optimization problems, and visualize the solution process step-by-step.

📘 Purpose

Automate simplex tableau creation, pivot operations, and final result interpretation for optimization-based mathematical problems.

📖 What is Simplex Method?

The Simplex Method is an algorithm used to solve linear programming problems by moving from one feasible solution to another until the optimal value is found.

🧮 How to Use

  • Paste or type your optimization problem
  • Or use the manual simplex grid
  • Choose Maximize / Minimize
  • Run the solver
  • View iterations and export report

⚙️ Supported Operations

  • Linear Programming Problems
  • Maximization / Minimization
  • Simplex & Big-M Method
  • Step-by-step iterations
  • History logs and PDF export

🚫 Limitations

  • Only supports linear optimization problems
  • OCR / image parsing is UI-ready but not fully connected
  • Advanced 3D visualization can be added next
